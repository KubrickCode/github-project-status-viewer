Privacy Policy

Last updated: 2025-10-09

Data Collection

This extension collects the following data:

  • GitHub OAuth access token (for API authentication)
  • GitHub username (for display purposes)

Data Usage

Collected data is used solely to:

  • Authenticate with GitHub API
  • Fetch project status information
  • Display status badges in issue lists

Data Storage

All data is stored in session storage and automatically deleted when you close your browser. No data is permanently stored or transmitted to third-party servers.

Data Sharing

We do not share, sell, or transmit your data to any third parties except GitHub API for functionality purposes.
